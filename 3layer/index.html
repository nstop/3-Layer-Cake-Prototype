3-Layer-Cake Prototype

A framework for creating multi-layered, annotated, web-based protoypes using HTML, CSS and JavaScript.

Wireframe > Design > Annotations

Create web prototypes in code that enable easy design view switching between wireframe, the full design and the annotated design.

Use the 3-Layer-Cake Prototype framework to insert dynamically-numbered annotations and a corresponding notes panel that provide in-browser documentation of content and functionality for clients and business stakeholders.
